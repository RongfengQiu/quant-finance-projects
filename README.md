## 1. Project Goal

This repository is a **portfolio-style end-to-end data science project**.

**Objective:** Evaluate the robustness of **Monte Carlo (MC)** option pricing compared with **Black–Scholes (BS)** using historical stock price data.

> Priority: **clean workflow + modular code + clear evaluation**, not complex financial modeling.

## 2. Industry Workflow (What this project demonstrates)

1. **Problem definition**
2. **Data loading** (CSV → DataFrame)
3. **Data cleaning** (dates, missing values, ordering)
4. **EDA** (notebook)
5. **Feature engineering** (returns, volatility)
6. **Train/Validation/Test split** (chronological, no look-ahead bias)
7. **Modeling** (BS benchmark + MC simulation)
8. **Evaluation & insights** (MAE/RMSE + SQL queries + plots + dashboard)

## 5. How to Run (Windows / Mac / Linux)

### 5.1 Install dependencies

```bash
python -m pip install -r requirements.txt
```

### 5.2 Run the end-to-end pipeline

From the project root:

```bash
python pipeline.py --recompute
```

Optional flags:

- Change MC simulations:

```bash
python pipeline.py --recompute --n-sim 10000
```

- Make it faster by sampling fewer dates:

```bash
python pipeline.py --recompute --sample-every 60
```

- Run a simple ML baseline (Ridge regression) to demonstrate a classic DS “model training” step:

```bash
python pipeline.py --recompute --run-ml
```

### 5.3 Launch dashboard

```bash
# Run from the project root directory
streamlit run dashboard/streamlit_app.py
```

Open the local URL shown in the terminal.
