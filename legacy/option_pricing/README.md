# Legacy Option-Pricing Extension

This folder keeps earlier finance modules from the previous version of the project.
They are **not part of the current main machine-learning pipeline**.

The current repository focus is:
- forecast future 20-day annualized realized volatility
- use a chronological ML workflow
- compare simple ML models against a naive volatility baseline

The files in this folder are kept only as archived finance extensions.

These archived files are kept for reference only. They are separated from the active `src/` pipeline and are not required to run the current ML project.
